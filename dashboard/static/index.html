Dashboard Kebakaran Hutan Los Angeles 2025

Dashboard ini memberikan visualisasi data yang menggambarkan titik api kebakaran di wilayah Los Angeles pada periode waktu antara 6 Januari 2025 hingga 30 Januari 2025. Titik api dideteksi menggunakan VIIRR (Visible Infrared Imaging Radiometer Suite) pada satelit NOAA-21 (JPSS-2) dengan resolusi sekitar 375 meter. Data yang didapatkan divisualisasikan dalam berbagai bentuk chart dan gambar, serta memungkinkan pengguna untuk menampilkan data berdasarkan tanggal tertentu.

Dashboard ini meliputi:

  • Tabel Lokasi dan Confidence Level, menampilkan tabel yang berisi informasi lokasi-lokasi yang memiliki titik api. Tabel ini menunjukkan jumlah titik api berdasarkan tingkat confidence level untuk setiap lokasi pada periode waktu yang dipilih.
  • Peta Interaktif Menampilkan titik api di wilayah Los Angeles, diwakili dengan marker berwarna yang menggambarkan tingkat confidence level masing-masing titik api.
  • Jumlah Confidence Berdasarkan Wilayah , berdasarkan wilayah menampilkan jumlah titik api dalam rentang waktu yang telah ditentukan, dikelompokkan berdasarkan wilayah dan tingkat confidence level. (1 Jan - 30 Jan)
  • Grafik Garis Level Confidence Menampilkan grafik yang menggambarkan perubahan jumlah titik api berdasarkan confidence level sepanjang waktu. Data ini dapat ditampilkan berdasarkan periode waktu yang dipilih pengguna.
  • Distribusi Level Tingkat Confidence, memvisualisasikan jumlah titik api untuk setiap confidence level berdasarkan periode waktu yang dipilih oleh pengguna.
